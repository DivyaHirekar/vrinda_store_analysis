# 📊 Vrinda Store Sales Dashboard (2022)

## 🧾 Objective

Vrinda Store is an e-commerce business that sells products across various categories through multiple online platforms. The objective of this project is to analyze the annual sales data for the year 2022 and create an interactive dashboard that provides meaningful insights into sales performance, customer behavior, and business trends.

The goal is to help the business make data-driven decisions by identifying key patterns such as top-performing categories, sales channels, and customer segments.

---

## 🛠️ Approach

### 1. Data Cleaning

* Removed duplicate entries
* Handled missing/null values
* Standardized data formats (dates, currency, categories)
* Corrected inconsistent values (e.g., gender labels, category names)

### 2. Data Processing

* Created new columns where required (e.g., Month, Age Group,Average order value)
* Converted data into structured format for analysis
* Ensured proper data types for calculations and visualization

### 3. Data Analysis

* Used Pivot Tables to summarize sales data
* Analyzed sales trends by month, category, and channel
* Compared performance across different customer segments

### 4. Dashboard Creation

* Built an interactive dashboard using Excel
* Added charts, slicers, and KPIs for better visualization
* Enabled easy filtering for dynamic insights

---

## 📈 Key Insights

* Women customers contributed more to overall sales compared to men
* Certain months showed higher sales, indicating seasonal trends
* Top-performing categories generated the majority of revenue
* Specific sales channels performed better than others
* Majority of orders came from a particular age group

---

## 📌 Conclusion

This dashboard provides a clear overview of Vrinda Store's sales performance in 2022. By leveraging these insights, the business can focus on high-performing segments, improve weaker areas, and plan strategies for future growth.
